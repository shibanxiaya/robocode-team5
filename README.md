# RoboCode Team5 - UltraRobot

## 项目简介

UltraRobot 是一个基于 Robocode 平台开发的高级战斗机器人，采用了多种先进的战斗策略和算法，在大地图（1200×1200）环境下表现出色。

## 核心特性

### 瞄准系统
- **圆周预测**：基于敌人运动轨迹的循环预测算法
- **转弯变化量追踪**：实时记录敌人转向变化
- **GF 统计偏移修正**：基于 Guassian Filter 的统计瞄准修正

### 走位系统
- **MRM (Minimum Risk Movement)**：最小风险移动算法
- **子弹波排斥**：智能躲避敌方子弹
- **大地图长距离移动**：优化的长距离移动候选点选择

### 雷达系统
- **即时收紧**：在 `onScannedRobot` 事件内快速锁定目标
- **主循环兜底全扫**：确保不会丢失目标

### 火力策略
- **分段火力**：根据距离和能量动态选择火力（扩展到 800+ 距离）
- **放宽开火阈值**：提高开火频率（从 6° 放宽到 10°）
- **近距击杀**：近距离高能量时的爆发输出

## v9 版本改动（1200×1200 大地图优化）

1. **移动候选距离扩大**：80/150/220/290 → 120/240/360/480
2. **火力分段扩展**：扩展到 800+ 距离，远距不再只打 1.2
3. **开火阈值调整**：6° → 10°，提高开火频率
4. **目标选择优化**：改为纯最近距离选择，混战乱战效率更高
5. **子弹波排斥系数微调**：避免过度躲避导致站桩

## 技术实现

### 关键算法

1. **圆周预测算法**：通过模拟敌人未来 90 个时间单位的运动轨迹来预测位置

2. **GF 统计瞄准**：使用 31 个统计桶记录敌人的运动模式，用于瞄准偏移修正

3. **风险评估移动**：综合考虑墙角风险、子弹波排斥、敌人距离等因素选择最优移动点

## 使用方法

1. 将 `UltraRobot.java` 复制到 Robocode 的 `robots/myrobots/` 目录下
2. 启动 Robocode，在机器人选择界面找到 `myrobots.UltraRobot`
3. 开始战斗！
